# Hướng dẫn chạy:

> clone repo tại: https://github.com/ducsiukap/-BTL-Digital-Image-Processing

1. Di chuyển tới folder chứa project sau khi clone:
   ```bash
   cd path/to/project
   ```
2. Chạy backend
   ```bash
   cd backend

   # cai thu vien can thiet
   pip install -r fastapi uvicorn imagio numpy python-multipart

   # chay backend
   uvicorn main:app
   ```
   > backend chạy tại: `localhost:8000`
3. Chạy frontend
    ```bash
    cd frontend

    # cai thu vien
    npm i

    # chay frontend
    npm run dev
    ```
    > frontend chạy tại: `localhost:5173`
